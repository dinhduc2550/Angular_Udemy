//INSTALL

2)Create folder have all html,css,ts file
  ng g c + [fileName]
3)Bootstrap3
  -b1: npm install --save bootstrap@3
  -b2: go to angular.json -> edit styles to add "node_modules/bootstrap/dist/css/bootstrap.min.css"
